# ExaBench Framework Documentation

## Overview

**ExaBench** is a benchmark framework for evaluating AI agent systems in High-Performance Computing (HPC) environments. It uses role-aware tasks, deterministic HPC state snapshots, and trace-based scoring.

See [01 — Overview](01-overview.md) for the single source of truth on principles and v0.1 scope.
